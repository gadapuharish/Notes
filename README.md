# Spring
A framework in java to build the backend ecosystem
Makes life easy in java coding and add dependency injection that free code load

**IOC(Inversion of control)**
We as a enterprise for out clients create a wonderful systems.

It comes down to 2 parts
- Business logic
- Object Creation

The object creation might seem simple, but if you merge 100 classes, it's not that simple.

_Spring_ is saying, hey you just focus on your business logic. I will take care of the object business, creating , managaing, optomising and even deletion.

As IOC concept, there are many ways to implement Dependency Injection.
